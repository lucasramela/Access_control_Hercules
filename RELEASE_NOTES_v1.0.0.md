# Hercules Gym v1.0.0

Primera version estable del sistema Hercules Gym.

## Incluye

- Control de acceso por DNI con pantalla dedicada para recepcion/entrada.
- Panel administrativo con resumen operativo.
- ABM de clientes con ficha digital.
- Gestion de planes con alta, edicion y eliminacion controlada.
- Cobro de cuotas, cambio de plan y registro de pagos.
- Prorrogas/promesas de pago para clientes con cuota vencida.
- Vencimientos con estados visuales por vencer/vencida.
- Caja diaria con apertura, movimientos, cierre e historial.
- Finanzas con filtros, pagos diarios y graficos dinamicos.
- Alertas con enlace directo a WhatsApp.
- Perfiles `admin` y `recepcion`.
- Base SQLite local.

## Usuarios iniciales

- Administrador: `admin` / `admin123`
- Recepcion: `recepcion` / `recepcion123`

## Importante

El archivo `gym_access.db` contiene datos reales y esta ignorado por Git. Para llevar datos a otra PC, copiar la base por fuera del repositorio o usar el instructivo de traslado.
